## Rekomendasi sistem

Node.js 18 / LTS terbaru

## jika belum mengaktifkan Yarn

```bash
corepack enable
```

## instalasi

```bash
git clone https://gitlab.com/sekaidotid/website-v3.git
```

```bash
cd website-v3
```

```bash
cp .env.example .env
```

```bash
yarn
```

```bash
yarn prepare
```

## deploy

```bash
yarn build
```
